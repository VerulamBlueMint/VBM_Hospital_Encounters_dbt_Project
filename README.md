# VBM Hospital Encounters (dbt + Fabric)

This repo contains a dbt project that connects to a Microsoft Fabric Warehouse and produces:

- `dbt_dev_silver.silver_encounters_final`
- `dbt_dev_gold.gold_encounters_final`
- `dbt_dev_gold.kpi_results`

See `Hospital_Encounters_dbt/README.md` for dbt setup and commands.
